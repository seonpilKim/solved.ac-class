[1181] 단어 정렬

문제
----
알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.

길이가 짧은 것부터
길이가 같으면 사전 순으로

입력
----
첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다. 주어지는 문자열의 길이는 50을 넘지 않는다.

출력
----
조건에 따라 정렬하여 단어들을 출력한다. 단, 같은 단어가 여러 번 입력된 경우에는 한 번씩만 출력한다.

풀이
----
 - Vector를 유일한 원소들의 집합으로 만드는 방법.
  1. sort : 알파벳 순 정렬
  2. unique : 연속된 중복 원소들을 vector의 끝으로 이동(쓰레기 값)
  3. erase : 중복된 원소들이 모여있는 끝부분을 삭제

위의 절차를 거쳐 vector 내의 원소들은 알파벳 순서를 기준으로 재배치 되었으며, 중복된 원소들은 제거되었다.
다시 길이가 짧은 순서로 재배치하기 위해 User-defined sort function을 추가하여 sort 함수를 호출하였다.

추가로 c++의 sort 함수는 unstable 방식으로 알파벳 순 정렬 이후 길이정렬을 하면, 같은 길이 내에서 알파벳 정렬이 유지가 되지 않을 수 있다.
이를 방지하기 위해 stable_sort 함수를 사용하였다.
(동일한 정렬 기준을 가진 원소들의 순서가 정렬 후에도 바뀌지 않으면 stable하다고 한다.)

참고로 merge sort, insertion sort가 대표적인 stable sort 알고리즘이고 unstable sort의 대표적인 알고리즘은 quick sort이다.
