[2751] 수 정렬하기 2

문제
----
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

입력
----
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄부터 N개의 줄에는 숫자가 주어진다. 이 수는 절댓값이 1,000,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

출력
----
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

풀이
----
<algorithm> 헤더를 include하여 sort() 함수를 사용하면 편리하다.
sort() 함수는 quick sort 기반으로, heap sort와 insertion sort를 섞은 방식이다.
최악의 경우 O(n^2)인 quick sort와 달리 O(nlogn)을 보장한다.
 - 1번째 인자 : 배열 pointer
 - 2번째 인자 : 배열 pointer + 배열 size
내림차순으로 정렬할 경우, 3번째 인자에 desc를 추가해주면 된다.
