[11726] 2xN 타일링 (https://www.acmicpc.net/problem/11726)

시간 제한	메모리 제한	제출	정답	맞은 사람	정답 비율

문제
----
2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

입력
----
첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)

출력
----
첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.

풀이
----
처음엔 BFS로 풀까 생각했었지만, n이 최대 1000까지 가능하므로 시간적으로 비효율이라는 생각이 들었다.

그래서 Dynamic Programming으로 접근해보았다.

타일을 배치하는 경우의 수는 다음 두가지로 나뉘어진다.
 1. 처음에 1x2 타일 1개를 배치 : 세로 1칸만 차지하므로 n-1개의 경우의 수만 따지면 됨.
 2. 처음에 2x1 타일 2개를 배치 : 세로 2칸을 차지하므로 n-2개의 경우의 수만 따지면 됨.

즉, 점화식은 다음과 같다.
dp[n] = dp[n-1] + dp[n-2] (dp[1] = 1, dp[2] = 2)
