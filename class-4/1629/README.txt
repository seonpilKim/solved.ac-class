[1629] 곱셈 (https://www.acmicpc.net/problem/1629)

문제
----
자연수 A를 B번 곱한 수를 알고 싶다. 단 구하려는 수가 매우 커질 수 있으므로 이를 C로 나눈 나머지를 구하는 프로그램을 작성하시오.

입력
----
첫째 줄에 A, B, C가 빈 칸을 사이에 두고 순서대로 주어진다. A, B, C는 모두 2,147,483,647 이하의 자연수이다.

출력
----
첫째 줄에 A를 B번 곱한 수를 C로 나눈 나머지를 출력한다.

풀이
----
단순히 반복문으로 하나하나 곱해가는 반복문으로는 곱셈 계산을 최대 2,147,483,647번 수행해야 하므로 Time Limit Error를 유발한다.
 - Time Complexity : O(n)

 - 자료형 범위
int : ±2,147,483,647
long long : ±9,223,372,036,854,775,807
(2,147,483,647)^2 = 4,611,686,014,132,420,609 < long long
최대값으로 최대 2제곱까지 가능하므로, 거듭제곱을 분할정복함으로써 해결할 수 있다.

 * 거듭제곱 분할정복
 : A에 A를 제곱한 값을 C로 나눈 나머지를 대입하고, B를 2로 나누는 방식.
 - Time Complexity : O(log n)

B가 홀수인 경우, 한 번 더 A를 곱해야하므로 먼저 r에 A값을 대입해주고, 
이후 A의 거듭제곱과 r을 곱하고 C로 나눈 나머지를 r에 저장한다. (r은 추후 출력할 값임)
B가 짝수인 경우, 추후 B가 1이되는 순간에 값을 r변수에 저장하여 출력하게 된다.

p.s) 추가로 비트연산자를 활용하여 계산속도를 향상시켰으며, 재귀함수대신 while문을 사용하여 불필요하게 스택을 많이 사용하는 상황을 피하였음.
